Remote Robotics

遠隔支援ロボット・機械腕開発

遠隔支援ロボット（経営革新テーマ）

LeRobot / XLeRobotを基盤として、小型自走台車と制御系を追加し、 家庭内支援で実用可能なロボットを目指しています。高齢者や身体の不自由な方への 日常支援を想定し、遠隔操作と安全性を両立するアーキテクチャを採用しています。

オープンソース互換性を維持しつつ、低価格構成（全体約660ドル）を志向し、 導入しやすい開発モデルを構築しています。

機械腕（双腕）システム

Raspberry Pi + MCU、カメラ、距離/IMUセンサーを統合し、WebRTC/ROSを介した 遠隔操作を実現します。物品移動や簡易作業を視野に、制御安定性と可搬性を重視した 構成へ発展させています。

3Dプリント依存構成の課題を踏まえ、耐久性・安全性・保守性を重視した ハードウェア設計および制御設計を進めています。

Product

ClearanceGate

高リスク実行の直前に配置する、形式的認可（Authorization）基盤

ClearanceGateは、意思決定や予測を行うシステムではなく、 「実行を許可できるか」を明示的ルールで判定する認可基盤です。 ワークフロー自体を置き換えるのではなく、既存システムの実行直前に配置して 認可境界を形成します。

判定結果は PROCEED / BLOCK / REQUIRE_ACK / DEGRADE の 4系統で決定され、不確実性がある場合は fail-closed（安全側）で制御します。 監査証跡を構造として残せる点を特長としています。

  • Deterministic outcome（決定論的な判定）
  • Fail-closed under uncertainty（安全側制御）
  • Auditability by design（監査可能性の構造化）
